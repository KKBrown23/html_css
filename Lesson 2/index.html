What is CSS

Css helps us to style our webpages. Css is used together with HTML to build websites.

How to use CSS:

  1. Create a CSS document in the same folder with the HTML document
  2. Connect CSS with HTML
    1. Use the link tag
    2. Add "rel" and "href" attributes to this tag
  3. Test if the CSS is connected to HTML
  4. Style the HTML using CSS

How to style an HTML element by CSS

  1. Select the HTML element at the CSS document
    • Selecting by tag`s name ex: h1
    • Selecting by class`s name ex: .btn
  2. Apply the styles that you need
