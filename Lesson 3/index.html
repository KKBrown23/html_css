The width of the elements in CSS

Block elements and Inline elements

Block elements have a width of 100% (they strech all the way), so they don`t allow any other element to sit in the same row with them. Some block elements are:

  • h1
  • p

Inline elements have a width equal to their content (they don`t strech all the way).Some inline elements are:

  • img
  • span
  • button

Spacing in CSS

What is spacing in Css

Spacing in CSS is about giving the elements of the page the right space inside or outside the element's borders. We include spacing because when elements are close to each other the page doesn`t look cool, also it`s hard to read.

Spacing outside the border. We use margin to put space outside the borders of the elements. We do this in order to seperate 2 elements from each other.

Spacing inside the border. We use padding to put space inside the borders of the element. We do this in order to make the element look bigger.
